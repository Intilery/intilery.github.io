The following events are reserved events in the Intilery Core Schema, you should not use these Event names/types except for their intended purpose.

These events can be found in the Intilery app on the Schema page shown below

Core Events

App​

Application Lifecycle Events​

  • Application Installed
  • Application Opened
  • Application Updated
  • Application Backgrounded
  • Application Crashed
  • Application Uninstalled

Campaign Events​

  • Push Notification Received
  • Push Notification Tapped
  • Push Notification Bounced
  • Install Attributed
  • Deep Link Clicked
  • Deep Link Opened
  • Install Attributed

Email​

  • Email Bounced
  • Email Delivered
  • Email Link Clicked
  • Email Marked as Spam
  • Email Opened
  • Unsubscribed

SMS​

  • SMS Sent
  • SMS Not Sent
  • SMS Delivered

API​

  • Identify
  • Track
  • Page
  • Screen

Subscription​

  • usubscribe from channel

Industry​

The following industry schemas can be added to your account within the Intilery CDP, enabling default events to be automatically recognised

  • Retail/Ecommerce
  • Gaming
  • Travel
  • Insurance
  • B2B
